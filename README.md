# Персональный дневник питания с анализом КБЖУ

Это простое веб-приложение для первой практической работы.

## Возможности
- добавление записи о приёме пищи;
- просмотр всех записей;
- расчёт суммарных калорий, белков, жиров и углеводов;
- модульная структура проекта.

## Структура проекта

```text
food-diary-practice-1/
├── client/
├── server/
│   ├── config/
│   ├── controllers/
│   ├── models/
│   ├── routes/
│   ├── services/
│   └── app.js
├── .env.example
├── .gitignore
├── package.json
└── README.md
```

## Запуск

```bash
npm install
cp .env.example .env
npm run dev
```

## Важное замечание
В проекте не используются жёстко прописанные локальные пути. Все настройки задаются через переменные окружения и относительные пути.
