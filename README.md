# 📋 Task Manager - Система управления рабочими процессами

Клиент-серверное веб-приложение для управления задачами и проектами с возможностью отслеживания статусов выполнения, распределения задач между исполнителями и анализа загрузки ресурсов.

## 🚀 Функциональные возможности

| Модуль | Функции |
|--------|---------|
| **Аутентификация** | Регистрация, вход, выход, управление сессиями |
| **Управление задачами** | CRUD операции, фильтрация по статусу/приоритету/исполнителю, поиск |
| **Управление проектами** | Создание проектов, группировка задач |
| **Аналитика** | Статистика по задачам, загрузка исполнителей, просроченные задачи |

## 🛠 Технологический стек

### Backend
| Технология | Назначение |
|------------|------------|
| **FastAPI** | Веб-фреймворк для REST API |
| **SQLAlchemy** | ORM для работы с базой данных |
| **Pydantic** | Валидация данных |
| **Uvicorn** | ASGI сервер |

### Клиентская часть
| Технология | Назначение |
|------------|------------|
| **Streamlit** | Веб-интерфейс |
| **Requests** | HTTP-запросы |
| **Rich** | Консольный клиент (CLI) |

### Инфраструктура
| Технология | Назначение |
|------------|------------|
| **Docker** | Контейнеризация |
| **Git** | Контроль версий |
